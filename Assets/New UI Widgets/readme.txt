- New UI Widgets - version 1.10.2

Widgets for new Unity UI.

Collections:
- Combobox
- ListView
- TileView
- TreeView
- DirectoryTreeView *
- FileListView *
Widgets can be generated for the your custom type with context menu "Create / New UI Widgets / Generate Widgets"

Containers:
- Accordion
- Tabs

Dialog:
- DateTimePicker
- DatePicker
- Dialog
- FileDialog *
- FolderDialog *
- Notify
- Picker
- Popup
- TimePicker

Input:
- Autocomplete
- Calendar
- ColorPicker
- ColorPickerRange
- Date
- DateTime
- RangeSlider
- Spinner
- Switch
- Time

Other:
- Paginator
- Progressbar
- Sidebar
- Tooltip
- Style (Skins) support

Supports:
- TextMesh Pro
- Data Bind for Unity

Interactions:
- Bring to Front
- Draggable
- Drag and Drop support
- Resizable

TreeView, TileView, Combobox, ListView (except ListViewGameObjects) optimized to use GameObjects only for visible nodes/tiles/items.

* not available on platforms with restricted access to file system (like WebGL and Windows Store/UWP)

Features
- Based on new UI
- Easy customizable
- Full source code included
- Fully documented
