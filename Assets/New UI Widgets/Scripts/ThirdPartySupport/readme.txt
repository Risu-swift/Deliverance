1. Data Bind for Unity support
	You can enable Data Bind for Unity support with Project Settings / New UI Widgets / Enable DataBind support.
	If Data Bind for Unity not installed option will not be available.

2. TextMesh Pro support
	You can enable TextMesh Pro support with Project Settings / New UI Widgets / Enable TextMesh Pro support.
	The required package will be installed automatically.
	If TextMesh Pro not installed option will not be available.
